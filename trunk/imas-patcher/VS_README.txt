To use these vcproj and sln files you must have the following installed:

- Visual Studio 2005 Professional or above (No express editions will work for this project as they do not support add-ins)
- Qt 4.5.2 SDK
- Qt Visual Studio Integration

To setup Visual Studio integrations with Qt, it is reccomended to read this article as they provide
a very good tutorial to use Qt with Visual Studio

http://dcsoft.com/community_server/blogs/dcsoft/archive/2009/03/06/how-to-setup-qt-4-5-visual-studio-integration.aspx

Static Linking:

To make static builds of the imas-patcher, add the -static switch to the configure command when compiling the Qt toolkit
